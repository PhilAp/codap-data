Attribute Key
  • manufacturer = manufacturer or division name
  • Car Line = model name
  • Displacement = displacement in litres
  • Num Cyl = number of engine cylinders
  • Transmission = transmission type
  • Drv Sys = drive axel type (Front Wheel Drive, Rear Wheel Drive, 4 Wheel Drive)
  • Index Num = index number
  • City MPG (mi/gal) = estimated city mileage per gallon
  • Hwy MPG (mi/gal) = estimated highway mileage per gallon
  • Comb MPG (mi/gal) = estimated combined mileage per gallon
  • Unrnd City EPA (mi/gal) = unadjusted city mileage per gallon
  • Unrnd Hwy EPA (mi/gal) = unadjusted highway mileage per gallon
  • Unrnd Comp EPA (mi/gal) = unadjusted combined mileage per gallon
  • Fuel Type = type of fuel the car model uses (Regular, Premium, Diesel, Compressed, Ethanol)
  • GUZLR = if the car model is a gas guzzler
  • Turbo if the car model is equipped with a turbocharger
  • SPCHGR = if the car model is equipped with a supercharger
  • AVG 2D Pass Vol (ft^3) = average passenger interior volume of a 2-door version
  • AVG_2D_LUGG_Vol (ft^3) = average luggage interior volume of a 2-door version
  • AVG_4D_PAS_Vol (ft^3) = average passenger interior volume of a 4-door version
  • AVG_4D_LUGG_Vol (ft^3) = average luggage interior volume of a 4-door version
  • AVG_HB_PAS_Vol (ft^3) = average passenger interior volume of a hatchback version
  • AVG_HB_LUGG_Vol (ft^3) = average luggage interior volume of a hatchback version
  • ANL_FL_CST = annual fuel cost based on estimated combined mpg
  • ENG_BLK_TXT = an engine related descriptor to identify a model from another similar model
  • Trans_DESC_TXT = a transmission related descriptor to identify a model from another similar model
  • VLVS_PER_CYL = # valves per cylinder (for 2000 or later only)
