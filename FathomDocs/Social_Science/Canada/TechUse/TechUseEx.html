This case table contains information about technology use in Canada from 2001. Some attributes are explained below.

Attribute Key:
  • Household = household size
  • Neighborhood = the type of neighborhood the person lives in, rural or urban
  • Connection? = if they are connected to the internet
  • Num_Computers = number of computers in the home
  • Num_Persons = number of persons who use the computer
  • Used_Computer = if they have used the computer in the last 12 months
  • Used_Email = if they have used email in the last 12 months
  • Spreadsheet = if they have used a spreadsheet in the last 12 months
  • Educational_CD = if they have used a type of Educational CD in the last 12 months
  • elem_and_hs = years of elementary and high school education
  • Friend_Family_importance = the importance of friends and family while learning how to use the computer
  • trial_error_importance = the importance of trial and error while self teaching how to use the computer
  • Hours_home_last_wk = number of hours of computer use the previous week
  • Schoolwork_hrs_changed = if the number of hours spent of schoolwork changed due to internet usage
  • Limitations = limitations due to physical or mental disability
